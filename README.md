# crytrac
Display value of crypto-currency holdings. Prices fetched from [CryptoCurrency Market Capitalizations](https://api.coinmarketcap.com/v1/ticker/).

Example `~/.hodlings`:
```
# Comments
BTC: 0.143
ETH: 31
DOGE: 1337 # so crypto
```

Sample output:
```
ETH: 	$2595.05 -9.9%
BTC: 	$252.90 6.1%
DOGE: 	$1.76 -4.32%
TOTAL: 	$2849.71
```
